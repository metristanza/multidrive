# Multi-Drive Smart Copy

**Easily copy files between your Google Drive accounts.**

A Chrome extension for users with multiple Google accounts who need to transfer files between their different Google Drives — without downloading and re-uploading.

## Features

- ✅ **Server-side copy** — Files transfer directly between Drives (no bandwidth used)
- ✅ **Multiple accounts** — Add and manage several Google accounts
- ✅ **Folder copying** — Copy entire folder structures (paid version)
- ✅ **Simple interface** — Browse, select, copy
